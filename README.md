# Healer Raid Frames

Three indicators on Blizzard's raid and party frames. Active only in healer specs.

## Features

- **Healer Buffs** (top-right) — Your HoTs and shields on the unit, with a golden glow.
- **Defensive Cooldowns** (top-left) — Major defensive buffs on the unit, with a green glow.
- **Dispellable CC** (bottom-left) — Crowd control debuffs you can dispel, with a red glow.
